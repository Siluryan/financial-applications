# Modelo — seção «Antes de começar» (copiar em cada lab)

Use **três blocos separados**. Leitor confunde quando tudo vira uma lista só.

```markdown
## Antes de começar

| Tipo | O que é |
|------|---------|
| **Conhecimento** | Ideias que você precisa *entender* (módulo ou [`ebook/PRE-REQUISITOS.md`](../ebook/PRE-REQUISITOS.md)) |
| **Labs anteriores** | Passos que você já *executou* neste repo (cluster no ar, lab concluído) |
| **Ambiente** | Programas instalados, RAM, serviços rodando |

### Conhecimento (este lab)

- …

### Labs anteriores

- …

### Ambiente

- …
```

**Curso inteiro:** conhecimento base (HTTP, Docker, K8s, Python, SQL) → [`ebook/PRE-REQUISITOS.md`](../ebook/PRE-REQUISITOS.md).  
**Este lab:** só o que é específico da ferramenta ou do passo de hoje.
